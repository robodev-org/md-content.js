## Purpose
Define minimal steps to reproduce failures deterministically.

## Scope
- Covers: reproduction requirements and environment constraints.
- Does not cover: CI pipeline setup.

## Table of Contents
{:toc}

## Requirements
- Use a fixed input document.
- Record resolved options.
- Capture DiagnosticsReport JSON.
- Avoid environment-specific inputs.

## Minimal Reproduction Template
- Input markdown
- Options used
- Diagnostics report
- Expected vs. actual output
