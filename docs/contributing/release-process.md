## Purpose
Define the minimal release process and required checks.

## Scope
- Covers: pre-release checks and documentation updates.
- Does not cover: CI pipeline details.

## Table of Contents
{:toc}

## Release Checks
- Tests MUST pass.
- Typecheck MUST pass.
- Diagnostics examples MUST be validated.
- Changelog MUST be updated.
